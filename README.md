## 概要
スポーツナビの一球速報をテレビのボールカウント表示のようにするアプリ

**果たして2017年のCSファーストステージまでに間に合うのか**

## 詳細
野球中継のボールカウントワイプのようにパソコン画面の右下に表示。  
データはスポーツナビ(https://sports.yahoo.co.jp/) の一球速報を利用。  
15秒ごとに更新。  
横浜戦以外もできます
